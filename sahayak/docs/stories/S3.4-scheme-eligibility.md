### T3.4.2 — search_schemes tool with eligibility matching
- **DoD:** Takes user profile, returns matching schemes ranked. No matches → suggests broadening criteria.
- **Status:** ✅ Complete
- **Evidence:** `schemes.py:21-58` — Matching logic:
  1. **Keyword match:** Query tokens matched against name + benefits + eligibility (case-insensitive).
  2. **Gender filter:** Checks `gender_filter` field (e.g. "female").
  3. **Category filter:** Checks `category_filter` field (comma-separated: "sc,st,obc,bpl").
  4. **Age filter:** Validates against `min_age` / `max_age` range.
  5. Returns `SchemesResponse(schemes=[Scheme(name, ministry, benefits, eligibility, how_to_apply, url)])`.
- **Tests:** 6 test cases — keyword search, pension search, gender filter, age filter, no results, housing search.
